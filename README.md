# Education Improvement Institute Gamma Site

This repository contains a temporary static version of the public Education Improvement Institute website for hosting providers that do not support Node.js.

It keeps the static/Kopage-compatible deployment model while adding the newer public pages and material prepared for the Node.js version, including the affordable IT services page.

## Preview

Open `index.html` directly in a browser, or serve the directory with any static web server. All CSS, JavaScript, and images are local to this repository.

## Deploy

Upload the repository contents to the web root in Kopage or another static hosting tool. The homepage is `index.html`.

No Node.js, database, cron jobs, or server-side routes are required for this static public website.
