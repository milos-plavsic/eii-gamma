# EII Kopage Static Site

This directory contains a temporary static version of the public EII website for hosting providers that do not support Node.js.

## Preview

Open `index.html` directly in a browser. All CSS, JavaScript, and images are local to this directory.

## Deploy

Upload the contents of `eii-Kopage` to the web root in Kopage or another static hosting tool. The homepage is `index.html`.

No Node.js, database, cron jobs, or server-side routes are required for this static public website.
